# ทางลัดแป้นพิมพ์ / Keyboard Shortcuts Reference

**คู่มือนี้เหมาะสำหรับ**: ผู้ใช้ที่ต้องการใช้งานระบบอย่างรวดเร็ว

---

## ภาพรวม / Overview

ระบบรองรับทางลัดแป้นพิมพ์พื้นฐานสำหรับการใช้งานทั่วไป

**หมายเหตุ**: ทางลัดส่วนใหญ่เป็น**มาตรฐานของเว็บเบราว์เซอร์**

---

## การนำทางทั่วไป / General Navigation

### Tab / แท็บ

**การกด**: <kbd>Tab</kbd>

**ฟังก์ชัน**: เคลื่อนไปยัง Input ถัดไป

**ใช้เมื่อ**: กรอกฟอร์ม (การจอง, Login)

**ตัวอย่าง**:
1. กรอก "ชื่อ" → กด <kbd>Tab</kbd>
2. Cursor ไปที่ "เบอร์โทร"
3. กรอก "เบอร์โทร" → กด <kbd>Tab</kbd>
4. Cursor ไปที่ "อีเมล"

---

### Shift + Tab / ย้อนกลับ

**การกด**: <kbd>Shift</kbd> + <kbd>Tab</kbd>

**ฟังก์ชัน**: เคลื่อนไปยัง Input ก่อนหน้า

**ใช้เมื่อ**: ต้องการกลับไปแก้ไข Input ก่อนหน้า

---

### Enter / ส่งฟอร์ม

**การกด**: <kbd>Enter</kbd>

**ฟังก์ชัน**: ส่งฟอร์ม (Submit)

**ใช้เมื่อ**:
- กรอกฟอร์ม Login → กด <kbd>Enter</kbd> = Login
- กรอกฟอร์มจอง → กด <kbd>Enter</kbd> = Next
- ค้นหา → กด <kbd>Enter</kbd> = Search

**ข้อควรระวัง**: ตรวจสอบว่ากรอกข้อมูลครบก่อนกด <kbd>Enter</kbd>

---

### Escape / ปิด Dialog

**การกด**: <kbd>Esc</kbd>

**ฟังก์ชัน**: ปิด Dialog/Modal

**ใช้เมื่อ**:
- Dialog เปิดอยู่ → กด <kbd>Esc</kbd> = ปิด Dialog
- Dropdown เปิดอยู่ → กด <kbd>Esc</kbd> = ปิด Dropdown

**ตัวอย่าง**:
- เปิด Appointment Details Dialog → กด <kbd>Esc</kbd> → ปิด Dialog

---

## การรีเฟรชหน้าจอ / Refresh Page

### F5 / รีเฟรช

**การกด**: <kbd>F5</kbd>

**ฟังก์ชัน**: รีเฟรชหน้าจอ

**ใช้เมื่อ**:
- หน้าจอค้าง
- ข้อมูลไม่อัปเดต

---

### Ctrl+R / รีเฟรช (Windows/Linux)

**การกด**: <kbd>Ctrl</kbd> + <kbd>R</kbd>

**ฟังก์ชัน**: รีเฟรชหน้าจอ (เหมือน F5)

---

### Cmd+R / รีเฟรช (macOS)

**การกด**: <kbd>Cmd</kbd> + <kbd>R</kbd>

**ฟังก์ชัน**: รีเฟรชหน้าจอ (เหมือน F5)

---

### Ctrl+Shift+R / Hard Refresh (Windows/Linux)

**การกด**: <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd>

**ฟังก์ชัน**: รีเฟรชและล้าง Cache

**ใช้เมื่อ**: หน้าจอไม่อัปเดตหลังรีเฟรชปกติ

---

### Cmd+Shift+R / Hard Refresh (macOS)

**การกด**: <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd>

**ฟังก์ชัน**: รีเฟรชและล้าง Cache

---

## การนำทางในปฏิทิน / Calendar Navigation

### Arrow Keys / ลูกศร

**การกด**: <kbd>←</kbd> <kbd>→</kbd> <kbd>↑</kbd> <kbd>↓</kbd>

**ฟังก์ชัน**: เคลื่อนที่ในปฏิทิน (บาง Calendar library รองรับ)

**ตัวอย่าง**:
- <kbd>→</kbd>: ไปวันถัดไป
- <kbd>←</kbd>: ไปวันก่อนหน้า
- <kbd>↑</kbd>: ไปสัปดาห์ก่อนหน้า
- <kbd>↓</kbd>: ไปสัปดาห์ถัดไป

**หมายเหตุ**: ขึ้นอยู่กับ Calendar component ที่ใช้ (React Big Calendar)

---

## ทางลัดเฉพาะระบบ / System-Specific Shortcuts

**ปัจจุบัน**: ระบบยัง**ไม่มี**ทางลัดเฉพาะ

**อนาคต**: อาจเพิ่มทางลัดเช่น:
- <kbd>Ctrl</kbd> + <kbd>K</kbd>: เปิด Command Palette
- <kbd>Ctrl</kbd> + <kbd>N</kbd>: สร้างนัดหมายใหม่
- <kbd>Ctrl</kbd> + <kbd>S</kbd>: บันทึก (ถ้ากำลังแก้ไขอยู่)

---

## เคล็ดลับ / Tips

### สำหรับผู้ใช้ทั่วไป

**ใช้ Tab สำหรับกรอกฟอร์มเร็ว**:
- กรอกชื่อ → <kbd>Tab</kbd>
- กรอกเบอร์ → <kbd>Tab</kbd>
- กรอกอีเมล → <kbd>Enter</kbd> (ส่งฟอร์ม)

**ใช้ Esc ปิด Dialog เร็ว**:
- แทนที่จะคลิก X → กด <kbd>Esc</kbd>

---

### สำหรับเจ้าหน้าที่/Admin

**ใช้ F5 รีเฟรชเมื่อข้อมูลไม่อัปเดต**:
- แม้ระบบมี Real-time updates (SSE)
- บางครั้งอาจต้องรีเฟรชด้วยตนเอง

**ใช้ Ctrl+Shift+R ถ้า F5 ไม่ได้ผล**:
- ล้าง Cache พร้อมรีเฟรช
- แก้ปัญหาหน้าจอไม่อัปเดต

---

## ข้อจำกัด / Limitations

**ไม่รองรับ**:
- ❌ Custom keyboard shortcuts (เฉพาะระบบ)
- ❌ Keyboard shortcuts สำหรับ Bulk operations
- ❌ Keyboard shortcuts สำหรับ Calendar actions (เช่น Create appointment ด้วย keyboard)

**อนาคต**:
- ✅ เพิ่ม Custom shortcuts configuration
- ✅ Keyboard shortcuts สำหรับการทำงานที่ทำบ่อย

---

## สรุปทางลัดที่สำคัญ / Key Shortcuts Summary

| Shortcut | Function | When to Use |
|----------|----------|-------------|
| <kbd>Tab</kbd> | ไปต่อ | กรอกฟอร์ม |
| <kbd>Enter</kbd> | ส่ง | Submit form |
| <kbd>Esc</kbd> | ปิด | ปิด Dialog |
| <kbd>F5</kbd> | รีเฟรช | อัปเดตข้อมูล |
| <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>R</kbd> | Hard refresh | ล้าง Cache |
| <kbd>Ctrl</kbd>+<kbd>C</kbd> | Copy | คัดลอกข้อความ |
| <kbd>Ctrl</kbd>+<kbd>V</kbd> | Paste | วางข้อความ |

**macOS**: ใช้ <kbd>Cmd</kbd> แทน <kbd>Ctrl</kbd>
