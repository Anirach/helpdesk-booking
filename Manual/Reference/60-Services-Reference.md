## รายละเอียดแต่ละบริการ / Service Details

### 🔧 HARDWARE (ซ่อมฮาร์ดแวร์)

**Duration**: 30 นาที

**เหมาะสำหรับ**:
- ปัญหาที่เกี่ยวกับ**อุปกรณ์ทางกายภาพ**
- เครื่องคอมพิวเตอร์, เครื่องพิมพ์, อุปกรณ์ต่อพ่วง

**ตัวอย่างปัญหา**:
- คอมพิวเตอร์เปิดไม่ติด / ไฟไม่ติด
- คอมพิวเตอร์ดับเอง / รีสตาร์ทเอง
- พัดลมดังมาก / ร้อนเกินไป
- เปลี่ยนอะไหล่ (RAM, HDD, SSD)
- เครื่องพิมพ์ติดกระดาษ / พิมพ์ไม่ออก

**สิ่งที่ลูกค้าต้องเตรียม**:
- นำเครื่องคอมพิวเตอร์มาด้วย (ถ้าเป็นเครื่อง Desktop)
- จด Serial Number หรือ Model
- เตรียมข้อมูลสำรอง (ถ้าต้องเปลี่ยน HDD/SSD)

---

### 💻 SOFTWARE (ติดตั้งซอฟต์แวร์)

**Duration**: 30 นาที

**เหมาะสำหรับ**:
- ปัญหาที่เกี่ยวกับ**โปรแกรม**และ**ระบบปฏิบัติการ**

**ตัวอย่างปัญหา**:
- ติดตั้งโปรแกรม Office, Photoshop, ฯลฯ
- อัปเดต Windows / macOS
- ลบไวรัส / Malware
- โปรแกรมไม่ทำงาน / ค้าง / ช้า
- ตั้งค่าโปรแกรม

**สิ่งที่ลูกค้าต้องเตรียม**:
- ไฟล์ติดตั้งโปรแกรม (ถ้ามี)
- License key (ถ้าต้องใช้)
- รายละเอียดปัญหา (เกิดเมื่อไหร่, ทำอะไรก่อนเกิดปัญหา)

---

### 🌐 NETWORK (แก้ไขเครือข่าย)

**Duration**: 45 นาที (**นานที่สุด**)

**เหมาะสำหรับ**:
- ปัญหาที่เกี่ยวกับ**อินเทอร์เน็ต**และ**เครือข่าย**

**ตัวอย่างปัญหา**:
- WiFi ต่อไม่ได้ / ไม่เห็น WiFi
- Internet ช้า / ขาดๆ หายๆ
- ไม่สามารถเข้าเว็บไซต์บางเว็บ
- ตั้งค่า Router / Modem
- แชร์ไฟล์/เครื่องพิมพ์ในเครือข่าย

**ทำไมนาน**:
- ต้อง**วินิจฉัย**ปัญหา (อาจเป็นเครื่อง, Router, ISP)
- ต้อง**ทดสอบ**หลายขั้นตอน
- อาจต้อง**ติดต่อ ISP** (ถ้าเป็นปัญหาสายอินเทอร์เน็ต)

**สิ่งที่ลูกค้าต้องเตรียม**:
- ข้อมูล WiFi (ชื่อ, รหัสผ่าน)
- ข้อมูล ISP (บริษัทอินเทอร์เน็ต, Package)
- รายละเอียดปัญหา (Internet หายเมื่อไหร่, ช้าหรือขาดๆ หายๆ)

---

### 👤 ACCOUNT (จัดการบัญชี)

**Duration**: 15 นาที (**เร็วที่สุด**)

**เหมาะสำหรับ**:
- ปัญหาที่เกี่ยวกับ**บัญชีผู้ใช้**และ**การเข้าถึงระบบ**

**ตัวอย่างปัญหา**:
- ลืมรหัสผ่าน Windows / macOS / ระบบองค์กร
- สร้างบัญชีใหม่
- ปลดล็อกบัญชี (ถูกล็อกเพราะกรอกรหัสผ่านผิดหลายครั้ง)
- เปลี่ยนรหัสผ่าน
- ตั้งค่า Permissions

**ทำไมเร็ว**:
- ขั้นตอนชัดเจน
- ไม่ต้องวินิจฉัย
- ไม่ต้องใช้เครื่องมือพิเศษ

**สิ่งที่ลูกค้าต้องเตรียม**:
- Username (ถ้าจำได้)
- บัตรประชาชน (เพื่อยืนยันตัวตน)

## วิธีเลือกบริการ / Decision Tree

```
เริ่ม: คุณมีปัญหาอะไร?

┌─────────────────────────┐
│ ปัญหาเกี่ยวกับอะไร?      │
└───────┬─────────────────┘
        │
        ├─ เครื่อง/อุปกรณ์ → 🔧 HARDWARE
        │
        ├─ โปรแกรม → 💻 SOFTWARE
        │
        ├─ อินเทอร์เน็ต/WiFi → 🌐 NETWORK
        │
        └─ รหัสผ่าน/บัญชี → 👤 ACCOUNT
```

**ตัวอย่าง**:

| ปัญหา | บริการที่เหมาะสม |
|-------|-------------------|
| คอมพิวเตอร์เปิดไม่ติด | 🔧 HARDWARE |
| ติดตั้ง Microsoft Office | 💻 SOFTWARE |
| WiFi ต่อไม่ได้ | 🌐 NETWORK |
| ลืมรหัสผ่าน | 👤 ACCOUNT |
| เครื่องพิมพ์ติดกระดาษ | 🔧 HARDWARE |
| โปรแกรมค้าง | 💻 SOFTWARE |
| Internet ช้า | 🌐 NETWORK |
| สร้างบัญชีใหม่ | 👤 ACCOUNT |

## เคล็ดลับ / Tips

### สำหรับลูกค้า / For Customers

**เลือกบริการให้ถูก**:
- อ่านคำอธิบายให้ดี
- ถ้าไม่แน่ใจ → โทรถาม (ก่อนจอง)

**เตรียมข้อมูล**:
- อธิบายปัญหาให้ชัดเจน
- จดรายละเอียด (เกิดเมื่อไหร่, ทำอะไรก่อนเกิดปัญหา)

---

### สำหรับเจ้าหน้าที่ / For Staff

**เลือกงานที่ถนัด**:
- ถนัด Hardware → Pickup งาน HARDWARE
- ถนัด Network → Pickup งาน NETWORK

**เตรียมอุปกรณ์**:
- ดูตาราง "Tools Needed"
- เตรียมอุปกรณ์ก่อนไปพบลูกค้า
