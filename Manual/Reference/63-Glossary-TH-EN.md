# พจนานุกรมศัพท์ ไทย-อังกฤษ / Glossary (TH-EN)

**คู่มือนี้เหมาะสำหรับ**: ทุกคนที่ต้องการเข้าใจศัพท์ในระบบ

---

## A

**Account / บัญชี**
- ความหมาย: บัญชีผู้ใช้งานในระบบ
- ตัวอย่าง: "Account Management" (จัดการบัญชี)

**Admin / ผู้ดูแลระบบ**
- ความหมาย: ผู้ใช้ที่มีสิทธิ์จัดการระบบ
- ตัวอย่าง: "Admin can assign jobs" (Admin สามารถมอบหมายงานได้)

**Appointment / นัดหมาย**
- ความหมาย: การจองเวลาเพื่อรับบริการ
- ตัวอย่าง: "Create an appointment" (สร้างนัดหมาย)

**Assign / มอบหมาย**
- ความหมาย: การกำหนดงานให้เจ้าหน้าที่
- ตัวอย่าง: "Assign to staff" (มอบหมายให้เจ้าหน้าที่)

**Availability / ความพร้อม**
- ความหมาย: ช่วงเวลาที่สามารถรับงานได้
- ตัวอย่าง: "Check availability" (เช็คความพร้อม)

**Available / ว่าง, พร้อม**
- ความหมาย: สามารถรับงานได้
- ตัวอย่าง: "Available jobs" (งานว่าง)

---

## B

**Booking / การจอง**
- ความหมาย: การจองนัดหมาย
- ตัวอย่าง: "Online booking" (จองออนไลน์)

**Bulk / เป็นกลุ่ม, จำนวนมาก**
- ความหมาย: ดำเนินการหลายรายการพร้อมกัน
- ตัวอย่าง: "Bulk assign" (มอบหมายเป็นกลุ่ม)

---

## C

**Calendar / ปฏิทิน**
- ความหมาย: การแสดงเวลาและนัดหมายในรูปแบบปฏิทิน
- ตัวอย่าง: "View calendar" (ดูปฏิทิน)

**Cancel / ยกเลิก**
- ความหมาย: ยกเลิกนัดหมาย
- ตัวอย่าง: "Cancel appointment" (ยกเลิกนัดหมาย)

**Completed / เสร็จสิ้น**
- ความหมาย: นัดหมายที่ทำเสร็จแล้ว
- ตัวอย่าง: "Mark as completed" (ทำเครื่องหมายว่าเสร็จสิ้น)

**Confirmed / ยืนยันแล้ว**
- ความหมาย: นัดหมายที่มีเจ้าหน้าที่รับผิดชอบแล้ว
- ตัวอย่าง: "Confirmed appointment" (นัดหมายที่ยืนยันแล้ว)

**Customer / ลูกค้า**
- ความหมาย: ผู้ใช้บริการศูนย์คอมพิวเตอร์
- ตัวอย่าง: "Customer information" (ข้อมูลลูกค้า)

---

## D

**Dashboard / แดชบอร์ด**
- ความหมาย: หน้าจอแสดงภาพรวมของระบบ
- ตัวอย่าง: "Staff dashboard" (แดชบอร์ดเจ้าหน้าที่)

**Date / วันที่**
- ความหมาย: วันที่นัดหมาย
- ตัวอย่าง: "Select date" (เลือกวันที่)

**Duration / ระยะเวลา**
- ความหมาย: ระยะเวลาของบริการ
- ตัวอย่าง: "30 minutes duration" (ระยะเวลา 30 นาที)

---

## E

**Export / ส่งออก**
- ความหมาย: การดาวน์โหลดข้อมูล
- ตัวอย่าง: "Export to CSV" (ส่งออกเป็น CSV)

---

## H

**Hardware / ฮาร์ดแวร์**
- ความหมาย: อุปกรณ์คอมพิวเตอร์ทางกายภาพ
- ตัวอย่าง: "Hardware repair" (ซ่อมฮาร์ดแวร์)

---

## I

**Installation / การติดตั้ง**
- ความหมาย: การติดตั้งซอฟต์แวร์
- ตัวอย่าง: "Software installation" (ติดตั้งซอฟต์แวร์)

---

## L

**Limited / จำกัด**
- ความหมาย: บริการจำกัด (เจ้าหน้าที่ว่างน้อย)
- ตัวอย่าง: "Limited service" (บริการจำกัด)

**Login / เข้าสู่ระบบ**
- ความหมาย: การเข้าใช้งานระบบด้วยบัญชีผู้ใช้
- ตัวอย่าง: "Staff login" (เจ้าหน้าที่เข้าสู่ระบบ)

**Logout / ออกจากระบบ**
- ความหมาย: การออกจากระบบ
- ตัวอย่าง: "Click logout" (คลิกออกจากระบบ)

---

## N

**Network / เครือข่าย**
- ความหมาย: ระบบเครือข่ายอินเทอร์เน็ต
- ตัวอย่าง: "Network troubleshooting" (แก้ไขปัญหาเครือข่าย)

**Notification / การแจ้งเตือน**
- ความหมาย: ข้อความแจ้งเตือนในระบบ
- ตัวอย่าง: "Push notification" (แจ้งเตือนแบบ Push)

---

## O

**Open / เปิดให้บริการ**
- ความหมาย: ระบบเปิดให้บริการปกติ
- ตัวอย่าง: "System is open" (ระบบเปิดให้บริการ)

---

## P

**Pending / รอดำเนินการ**
- ความหมาย: นัดหมายที่รอเจ้าหน้าที่รับ
- ตัวอย่าง: "Pending confirmation" (รอยืนยัน)

**Pickup / รับงาน**
- ความหมาย: การรับงานด้วยตัวเอง (ไม่รอ Admin assign)
- ตัวอย่าง: "Pickup job" (รับงาน)

**Problem / ปัญหา**
- ความหมาย: ปัญหาที่ลูกค้าต้องการแก้ไข
- ตัวอย่าง: "Problem description" (คำอธิบายปัญหา)

---

## R

**Reassign / มอบหมายใหม่**
- ความหมาย: การเปลี่ยนเจ้าหน้าที่ที่รับผิดชอบ
- ตัวอย่าง: "Reassign to another staff" (มอบหมายให้เจ้าหน้าที่คนอื่น)

**Reference Number / เลขที่อ้างอิง**
- ความหมาย: รหัสอ้างอิงนัดหมาย (8 ตัวอักษรแรกของ ID)
- ตัวอย่าง: "Reference: A1234567" (อ้างอิง: A1234567)

**Report / รายงาน**
- ความหมาย: รายงานสถิติของระบบ
- ตัวอย่าง: "Monthly report" (รายงานประจำเดือน)

---

## S

**Service / บริการ**
- ความหมาย: ประเภทบริการที่ให้ (Hardware, Software, Network, Account)
- ตัวอย่าง: "Select service" (เลือกบริการ)

**Slot / ช่อง, ช่วงเวลา**
- ความหมาย: ช่วงเวลาที่สามารถจองได้
- ตัวอย่าง: "Time slot" (ช่วงเวลา)

**Software / ซอฟต์แวร์**
- ความหมาย: โปรแกรมคอมพิวเตอร์
- ตัวอย่าง: "Software installation" (ติดตั้งซอฟต์แวร์)

**Staff / เจ้าหน้าที่**
- ความหมาย: เจ้าหน้าที่ศูนย์คอมพิวเตอร์
- ตัวอย่าง: "Staff member" (สมาชิกเจ้าหน้าที่)

**Status / สถานะ**
- ความหมาย: สถานะของนัดหมาย
- ตัวอย่าง: "Appointment status" (สถานะนัดหมาย)

---

## T

**Time / เวลา**
- ความหมาย: เวลานัดหมาย
- ตัวอย่าง: "Select time" (เลือกเวลา)

**Troubleshooting / แก้ไขปัญหา**
- ความหมาย: การวิเคราะห์และแก้ไขปัญหา
- ตัวอย่าง: "Network troubleshooting" (แก้ไขปัญหาเครือข่าย)

---

## U

**Unavailability / เวลาไม่ว่าง**
- ความหมาย: ช่วงเวลาที่ไม่สามารถรับงานได้
- ตัวอย่าง: "Create unavailability" (กำหนดเวลาไม่ว่าง)

**Utilization / การใช้งาน**
- ความหมาย: อัตราการใช้เวลาทำงาน (%)
- ตัวอย่าง: "Utilization rate" (อัตราการใช้งาน)

---

## V

**View / ดู, แสดง**
- ความหมาย: การดูข้อมูล
- ตัวอย่าง: "View details" (ดูรายละเอียด)

---

## W

**Walk-in / ลูกค้าที่เดินเข้ามา**
- ความหมาย: ลูกค้าที่มาที่เคาน์เตอร์โดยตรง (ไม่จองออนไลน์)
- ตัวอย่าง: "Walk-in customer" (ลูกค้า Walk-in)

**Working Hours / เวลาทำการ**
- ความหมาย: ช่วงเวลาที่เปิดให้บริการ (08:30-16:30)
- ตัวอย่าง: "Standard working hours" (เวลาทำการมาตรฐาน)

**Workload / ภาระงาน**
- ความหมาย: ปริมาณงานของเจ้าหน้าที่
- ตัวอย่าง: "Staff workload" (ภาระงานเจ้าหน้าที่)

---

## คำศัพท์เฉพาะทาง / Technical Terms

**API / Application Programming Interface**
- ความหมาย: ตัวกลางที่ให้โปรแกรมสื่อสารกัน
- ไม่จำเป็นต้องรู้สำหรับผู้ใช้ทั่วไป

**CSV / Comma-Separated Values**
- ความหมาย: รูปแบบไฟล์ข้อมูลที่เปิดได้ใน Excel
- ใช้สำหรับ Export ข้อมูล

**PDF / Portable Document Format**
- ความหมาย: รูปแบบไฟล์เอกสาร
- ใช้สำหรับ Export รายงาน

**SSE / Server-Sent Events**
- ความหมาย: เทคโนโลยีส่งการแจ้งเตือนแบบเรียลไทม์
- ใช้ภายในระบบ
